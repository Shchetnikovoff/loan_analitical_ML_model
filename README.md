# Credit Risk Prediction - Case 9

## Setup
1.  **Data**: Download `train.csv` and `test.csv` with credit application data and place them in the `data/` folder.
2.  **Environment**: Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Running the Code
To train the model and generate a submission:
```bash
cd src
python train.py
```

## Project Structure
-   `data/`: Place dataset files here.
-   `src/`: Source code for preprocessing and modeling.
-   `notebooks/`: Jupyter notebooks for EDA and credit risk analysis.
-   `submission.csv`: Generated submission file.
-   `create_notebook.py`: Script to generate the Jupyter notebook.
